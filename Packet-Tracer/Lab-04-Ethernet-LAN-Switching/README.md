
# Lab 04 – Ethernet LAN Switching

## Objective
Learn how Ethernet LAN switching works and how switches forward data between devices using MAC addresses.

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI

## Devices Used
- Cisco 2960 Switch
- PCs
- Ethernet Cables

## Concepts Covered
- Ethernet LAN
- Switch operation
- MAC Address Table
- Frame forwarding
- Broadcast and Unicast traffic

## Commands Practiced

text
enable
show mac address-table
show interfaces
show running-config
show version


## Verification
- Connected multiple PCs to a switch.
- Verified that devices could communicate.
- Viewed the MAC address table using the show mac address-table command.
- Observed how the switch learned MAC addresses automatically.

## Skills Learned
- Understanding how Ethernet switches operate.
- Learning how switches build a MAC address table.
- Verifying switch operation using Cisco IOS commands.
- Identifying the difference between broadcast and unicast traffic.

## Outcome
Successfully explored Ethernet LAN switching and understood how switches forward frames based on MAC addresses.
