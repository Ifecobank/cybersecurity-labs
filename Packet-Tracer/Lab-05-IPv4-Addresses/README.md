# Lab 05 – IPv4 Addresses

## Objective
Learn the structure of IPv4 addresses, subnet masks, and how IP addresses are assigned to network devices.

## Tools Used
- Cisco Packet Tracer
- Cisco IOS CLI

## Concepts Covered
- IPv4 address
- Network ID
- Host ID
- Subnet mask
- Default gateway
- Public vs. Private IP addresses

## Example IP Addressing

| Device | IP Address | Subnet Mask | Default Gateway |
|---------|------------|-------------|-----------------|
| PC1 | 192.168.1.10 | 255.255.255.0 | 192.168.1.1 |
| PC2 | 192.168.1.20 | 255.255.255.0 | 192.168.1.1 |
| Router G0/0 | 192.168.1.1 | 255.255.255.0 | N/A |

## Commands Practiced

text
show ip interface brief
show running-config
ipconfig
ping


## Verification
- Assigned IPv4 addresses to network devices.
- Verified that the IP addresses were correctly configured.
- Successfully tested connectivity using the ping command.

## Skills Learned
- Understanding IPv4 address structure.
- Configuring IP addresses.
- Identifying network and host portions of an address.
- Verifying connectivity between devices.

## Outcome
Successfully configured IPv4 addresses and verified communication between devices using the correct IP addressing scheme.
