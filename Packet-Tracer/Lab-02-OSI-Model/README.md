OSI Model

## Objective
Understand the seven layers of the OSI (Open Systems Interconnection) Model and how data travels through a network.

## Tools Used
- Cisco Packet Tracer
- Cisco Networking Academy

## The Seven Layers of the OSI Model

| Layer | Name | Function |
|-------|------|----------|
| 7 | Application | Provides network services to applications such as web browsers and email clients. |
| 6 | Presentation | Translates, encrypts, and compresses data. |
| 5 | Session | Establishes, manages, and terminates communication sessions. |
| 4 | Transport | Ensures reliable data delivery using TCP or fast delivery using UDP. |
| 3 | Network | Routes packets between different networks using IP addresses. |
| 2 | Data Link | Uses MAC addresses to transfer data between devices on the same network. |
| 1 | Physical | Transmits data as electrical, optical, or radio signals through cables or wireless media. |

## Key Concepts Learned
- The OSI Model has seven layers.
- Each layer has a specific responsibility.
- Data is encapsulated as it moves down the layers and de-encapsulated as it moves up.
- Different networking devices operate at different OSI layers.

## Skills Gained
- Identifying the seven OSI layers.
- Understanding the purpose of each layer.
- Relating network devices and protocols to the correct OSI layer.

## Outcome
Successfully learned the functions of each OSI layer and how they work together to enable network communication.
