# Lab 10 – Multi-Switch Network

## Objective

Configure and verify communication in a network with multiple switches, ensuring devices connected to different switches can communicate successfully.

## Tools Used

- Cisco Packet Tracer
- Cisco IOS CLI

## Devices Used

- Cisco 2960 Switches
- PCs
- Routers (if applicable)
- Copper Straight-Through Cables

## Concepts Covered

- Multi-switch topology
- Switch-to-switch connections
- MAC Address Table
- Ethernet Switching
- Frame Forwarding
- End-to-End Connectivity
- Basic Network Troubleshooting

## Lab Activities

- Connected multiple switches.
- Connected PCs to different switches.
- Configured IP addresses on all PCs.
- Verified switch interfaces.
- Tested communication between devices.
- Observed MAC address learning.

## Commands Used

```text
show mac address-table
show interfaces
show running-config
show ip interface brief
ping
```

## Verification

- Verified all switch ports were operational.
- Confirmed MAC addresses were learned dynamically.
- Successfully communicated between devices connected to different switches.
- Verified network connectivity using ping.

## Skills Learned

- Connecting multiple switches.
- Understanding how switches forward frames.
- Viewing and interpreting the MAC address table.
- Troubleshooting Layer 2 connectivity issues.
- Verifying end-to-end communication.

## Outcome

Successfully configured a multi-switch network and verified communication between devices connected through different switches.
