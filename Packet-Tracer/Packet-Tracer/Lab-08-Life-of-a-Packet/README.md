# Lab 08 – Life of a Packet

## Objective

Understand how a packet travels across a network and how devices process data from the source to the destination.

## Tools Used

- Cisco Packet Tracer
- Simulation Mode

## Devices Used

- PCs
- Cisco Switches
- Cisco Routers

## Concepts Covered

- Packet Encapsulation
- Packet De-encapsulation
- OSI Model
- TCP/IP Model
- MAC Address
- IP Address
- ARP
- Ethernet Frames
- Routing

## Lab Activities

- Built a simple network topology.
- Sent a packet from one PC to another.
- Used Packet Tracer Simulation Mode to observe the packet's journey.
- Examined how data moved through each OSI layer.
- Observed ARP requests and replies.
- Watched routers forward packets based on IP addresses.
- Verified successful delivery to the destination.

## Commands Used

text
show ip interface brief
show ip route
ping
arp -a


## Verification

- Successfully sent packets between devices.
- Observed packet encapsulation and de-encapsulation.
- Verified ARP resolution.
- Confirmed successful packet delivery.

## Skills Learned

- Understanding the life cycle of a network packet.
- Following packets through the OSI Model.
- Distinguishing between MAC addresses and IP addresses.
- Understanding how switches and routers forward traffic.
- Using Packet Tracer Simulation Mode for troubleshooting.

## Outcome

Successfully analyzed the journey of a packet through a network and gained a better understanding of how data is transmitted between devices.
