
# Lab 01 – Connecting Devices in Cisco Packet Tracer

## Objective
Learn how to connect networking devices in Cisco Packet Tracer using the correct cable types.

## Tools Used
- Cisco Packet Tracer

## Devices Used
- Router (Cisco 2911)
- Switch (Cisco 2960)
- PCs

## Connections Made
| Device 1 | Device 2 | Cable Used |
|----------|----------|------------|
| PC | Switch | Copper Straight-Through |
| Switch | Router | Copper Straight-Through |
| PC | Router (Console Port) | Console Cable (when configuring the router) |

## Skills Learned
- Identifying network devices.
- Selecting the correct cable type.
- Connecting devices properly.
- Understanding interface names such as G0/0 and G0/1.

## Challenges
- Learned that using the wrong cable can prevent devices from communicating.
- Learned how to identify the correct ports.

## Outcome
Successfully connected all devices and prepared the network for configuration in future labs
